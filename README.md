## 项目结构

```
src/
├── main.ts                    # 入口文件（切换课程在这里改 import）
├── style.css                  # 全局样式
├── core/                      # 可复用核心模块
│   └── SceneManager.ts        # 场景管理器（第 1 课产物）
├── lessons/                   # 每课代码
│   ├── 01-project-architecture/
│   │   └── main.ts
│   ├── 02-camera-controls/
│   │   └── main.ts
│   └── ...
└── assets/                    # 纹理、模型等资源
```

## 常用命令

```bash
pnpm dev        # 启动开发服务器 (http://localhost:3300)
pnpm build      # 构建生产版本
```
